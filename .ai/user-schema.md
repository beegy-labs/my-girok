# User Schema

> Universal user schema for 15+ service types | SCIM 2.0 + JSONB extensions

| Category    | Service Types      | Key Roles                   |
| ----------- | ------------------ | --------------------------- |
| Commerce    | Marketplace, B2B   | BUYER, SELLER, PURCHASER    |
| Social      | SNS, Chat, Gaming  | USER, CREATOR, MODERATOR    |
| Content     | Blog, Streaming    | READER, AUTHOR, EDITOR      |
| Enterprise  | LMS, Fintech, SaaS | LEARNER, INSTRUCTOR, OWNER  |
| Operations  | Logistics, Booking | DRIVER, HOST, DISPATCHER    |
| Specialized | Healthcare, IoT    | PATIENT, DOCTOR, HOME_OWNER |

**Permission Format**: `{resource}:{action}:{scope}`
**SSOT**: `docs/llm/policies/user-schema.md`
