# Internationalization (i18n)

react-i18next with Korean default, English fallback

## Languages

| Code | Language | Role     |
| ---- | -------- | -------- |
| ko   | Korean   | Default  |
| en   | English  | Fallback |
| ja   | Japanese | Optional |

## Structure

```
apps/web-girok/src/i18n/
  config.ts
  locales/{ko,en,ja}.json
```

## Config

```typescript
i18n.use(initReactI18next).init({
  resources,
  lng: localStorage.getItem('language') || 'ko',
  fallbackLng: 'en',
  interpolation: { escapeValue: false },
});
```

## Usage

```typescript
import { useTranslation } from 'react-i18next';
const { t, i18n } = useTranslation();

<h1>{t('home.title')}</h1>

i18n.changeLanguage('en');
localStorage.setItem('language', 'en');
```

## Namespaces

| Namespace | Purpose              |
| --------- | -------------------- |
| common    | save, cancel, delete |
| nav       | login, register      |
| auth      | Auth forms           |
| resume    | Resume forms/lists   |
| errors    | Error messages       |

## Interpolation

```typescript
t('welcome', { name: user.name }); // "Welcome, {{name}}!"
t('duration', { years: 2, months: 3 }); // "{{years}} years {{months}} months"
```

## Rules

| Do                          | Don't                      |
| --------------------------- | -------------------------- |
| Add to ALL languages        | Add to just one            |
| Use hierarchical keys       | Use flat keys like `text1` |
| Use `common.*` for reusable | Duplicate translations     |
| Test all languages          | Skip testing               |
