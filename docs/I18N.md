# Internationalization (i18n)

> react-i18next with Korean default, English fallback

## Languages

| Code | Language | Role     |
| ---- | -------- | -------- |
| ko   | Korean   | Default  |
| en   | English  | Fallback |
| ja   | Japanese | Optional |

## Structure

```
apps/web-main/src/i18n/
├── config.ts           # i18n initialization
└── locales/
    ├── ko.json
    ├── en.json
    └── ja.json
```

## Configuration

```typescript
i18n.use(initReactI18next).init({
  resources,
  lng: localStorage.getItem('language') || 'ko',
  fallbackLng: 'en',
  interpolation: { escapeValue: false },
});
```

## Usage

```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t, i18n } = useTranslation();

  return <h1>{t('home.title')}</h1>;
}

// Change language
i18n.changeLanguage('en');
localStorage.setItem('language', 'en');
```

## Namespaces

| Namespace | Purpose                              |
| --------- | ------------------------------------ |
| common    | save, cancel, delete, loading, error |
| nav       | login, register, logout              |
| home      | Home page content                    |
| auth      | Authentication forms                 |
| resume    | Resume forms and lists               |
| settings  | Settings page                        |
| errors    | Error messages                       |

## Interpolation

```typescript
// Translation: "Welcome, {{name}}!"
t('welcome', { name: user.name });

// Translation: "{{years}} years {{months}} months"
t('duration', { years: 2, months: 3 });
```

## Adding Translations

1. Add key to ALL language files (ko, en, ja)
2. Use in component: `{t('newKey')}`
3. Test all three languages

## Best Practices

| Do                          | Don't                      |
| --------------------------- | -------------------------- |
| Add to ALL languages        | Add to just one            |
| Use hierarchical keys       | Use flat keys like `text1` |
| Use `common.*` for reusable | Duplicate translations     |
| Test all languages          | Skip testing               |
