# 국제화 (i18n)

> react-i18next와 한국어 기본, 영어 대체 언어 설정

## 언어

| 코드 | 언어   | 역할      |
| ---- | ------ | --------- |
| ko   | 한국어 | 기본      |
| en   | 영어   | 대체 언어 |
| ja   | 일본어 | 선택 사항 |

## 구조

```
apps/web-main/src/i18n/
├── config.ts           # i18n 초기화
└── locales/
    ├── ko.json
    ├── en.json
    └── ja.json
```

## 설정

```typescript
i18n.use(initReactI18next).init({
  resources,
  lng: localStorage.getItem('language') || 'ko',
  fallbackLng: 'en',
  interpolation: { escapeValue: false },
});
```

## 사용법

```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t, i18n } = useTranslation();

  return <h1>{t('home.title')}</h1>;
}

// 언어 변경
i18n.changeLanguage('en');
localStorage.setItem('language', 'en');
```

## 네임스페이스

| 네임스페이스 | 목적                            |
| ------------ | ------------------------------- |
| common       | 저장, 취소, 삭제, 로딩 중, 오류 |
| nav          | 로그인, 회원가입, 로그아웃      |
| home         | 홈 페이지 내용                  |
| auth         | 인증 폼                         |
| resume       | 이력서 폼 및 목록               |
| settings     | 설정 페이지                     |
| errors       | 오류 메시지                     |

## 번역 키

```json
{
  "nav": {
    "login": "로그인"
  },
  "resume": {
    "myResumes": "내 이력서",
    "form": {
      "name": "이름",
      "email": "이메일"
    }
  }
}
```

## 보간법

```typescript
// 번역: "Welcome, {{name}}!"
t('welcome', { name: user.name });

// 번역: "{{years}} years {{months}} months"
t('duration', { years: 2, months: 3 });
```

## 번역 추가

1. 모든 언어 파일(ko, en, ja)에 키를 추가합니다.
2. 컴포넌트에서 사용합니다: `{t('newKey')}`
3. 세 가지 언어를 모두 테스트합니다.

## 모범 사례

| 해야 할 일                           | 하지 말아야 할 일           |
| ------------------------------------ | --------------------------- |
| 모든 언어에 추가                     | 단 하나만 추가              |
| 계층적 키 사용                       | `text1`과 같은 평면 키 사용 |
| 재사용 가능한 내용은 `common.*` 사용 | 번역 중복                   |
| 모든 언어 테스트                     | 테스트 생략                 |
