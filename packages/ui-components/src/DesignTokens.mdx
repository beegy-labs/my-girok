# Design Tokens

**V0.0.1 AAA Workstation** - WCAG 2.1 AAA Compliant Color System

All colors are defined in `@my-girok/design-tokens/tokens.css` and exposed via Tailwind CSS 4 `@theme` inline.

> **Note**: Use the Theme toggle in the toolbar to preview colors in Light and Dark mode.

## Typography Tokens

### Font Families

<Typeset

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--font-family-serif-title` | Playfair Display, Georgia, serif | `font-serif-title` | Editorial headings |
| `--font-family-mono-brand` | ui-monospace, SFMono-Regular, monospace | `font-mono-brand` | Brand labels, code |

### Letter Spacing

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--letter-spacing-brand-sm` | 0.2em | `tracking-brand-sm` | Small labels |
| `--letter-spacing-brand-md` | 0.25em | `tracking-brand-md` | Footer links |
| `--letter-spacing-brand` | 0.3em | `tracking-brand` | Default brand text |
| `--letter-spacing-brand-lg` | 0.5em | `tracking-brand-lg` | Large brand displays |
| `--letter-spacing-editorial` | -0.05em | `tracking-editorial` | Serif headings (tight) |

## Usage Examples

```tsx
// SSOT Pattern - ALWAYS use token-based utilities
<h1 className="font-serif-title tracking-editorial italic text-theme-text-primary">
  Page Title
</h1>

<div className="rounded-editorial-lg bg-theme-bg-card border border-theme-border-subtle p-10 shadow-theme-md">
  Form Card
</div>

<span className="font-mono-brand tracking-brand uppercase text-theme-text-accent">
  GIROK.
</span>

<button className="
  bg-gradient-to-r from-btn-primary-from to-btn-primary-to
  text-btn-primary-text rounded-input px-6 py-3
  focus-visible:ring-4 focus-visible:ring-theme-focus-ring
  transition-all duration-200 ease-editorial
">
  Primary Button
</button>
```

### Anti-Patterns (DO NOT USE)

```tsx
// BAD: Hardcoded pixel values
<div className="rounded-[48px]">

// BAD: Inline styles with CSS variables
<h1 style={{ fontFamily: 'var(--font-family-serif-title)' }}>

// BAD: Arbitrary tracking values
<span className="tracking-[0.3em]">

// BAD: Hardcoded colors
<div className="bg-[#6b4a2e]">
```

## CSS Variable Reference

All tokens are CSS custom properties. Access them directly when needed:

```css
/* In CSS */
.custom-element {
  background: var(--theme-bg-card);
  color: var(--theme-text-primary);
  border-radius: var(--border-radius-editorial);
}
```

```tsx
// In JavaScript/TypeScript
const bgColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--theme-bg-page');
```

## Maintenance

When updating design tokens:

1. **Edit source file**: `packages/design-tokens/src/tokens.css`
2. **Update this documentation**: Add new colors/tokens to tables above
3. **Update manager.ts**: If branding colors change, sync `GIROK_TOKENS` object
4. **Verify locally**: `pnpm --filter @my-girok/ui-components storybook`
5. **Check WCAG compliance**: Ensure all text meets 7:1+ contrast ratio

See [SSOT Strategy](?path=/docs/introduction--docs#ssot-utility-classes) for naming conventions.
