## Typography Tokens

### Font Families

| Token | Value | Tailwind |
|-------|-------|----------|
| `--font-family-serif-title` | Playfair Display, Georgia, serif | `font-serif-title` |
| `--font-family-mono-brand` | ui-monospace, SFMono-Regular, monospace | `font-mono-brand` |

### Letter Spacing

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--letter-spacing-brand-sm` | 0.2em | `tracking-brand-sm` | Small labels |
| `--letter-spacing-brand-md` | 0.25em | `tracking-brand-md` | Footer links |
| `--letter-spacing-brand` | 0.3em | `tracking-brand` | Default brand |
| `--letter-spacing-brand-lg` | 0.5em | `tracking-brand-lg` | Large brand |
| `--letter-spacing-editorial` | -0.05em | `tracking-editorial` | Serif headings |

---

## Spacing Tokens

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--spacing-nav` | 80px | `pt-nav`, `h-nav` | Navigation height |
| `--spacing-section` | 96px | `mb-section` | Section margin |
| `--spacing-footer-top` | 96px | `pt-footer-top` | Footer top padding |
| `--spacing-footer-bottom` | 128px | `pb-footer-bottom` | Footer bottom padding |

---

## Border Radius Tokens

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--border-radius-input` | 24px | `rounded-input` | Inputs, buttons |
| `--border-radius-widget` | 32px | `rounded-widget` | Widgets, modals |
| `--border-radius-editorial` | 40px | `rounded-editorial` | Cards |
| `--border-radius-editorial-lg` | 48px | `rounded-editorial-lg` | Form cards |
| `--border-radius-editorial-xl` | 56px | `rounded-editorial-xl` | Hero sections |
| `--border-radius-editorial-2xl` | 64px | `rounded-editorial-2xl` | Menu cards |

## Usage Examples

```tsx
// SSOT Pattern - ALWAYS use token-based utilities
<h1 className="font-serif-title tracking-editorial italic">
  Page Title
</h1>

<div className="rounded-editorial-lg bg-theme-bg-card border border-theme-border-subtle p-10">
  Form Card
</div>

<span className="font-mono-brand tracking-brand uppercase text-theme-text-accent">
  GIROK.
</span>

// Anti-Pattern - NEVER use hardcoded values
<div className="rounded-[48px]">  // BAD
<h1 style={{ fontFamily: 'var(--font-family-serif-title)' }}>  // BAD
<span className="tracking-[0.3em]">  // BAD
```

## Maintenance

When updating design tokens:

1. Edit `packages/design-tokens/src/tokens.css`
2. Update this documentation if new tokens are added
3. Run `pnpm --filter @my-girok/ui-components build-storybook` to verify
4. Ensure all components still meet WCAG AAA contrast requirements
